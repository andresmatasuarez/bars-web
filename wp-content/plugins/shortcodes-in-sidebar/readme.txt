Shortcodes in Sidebar allows shortcodes to execute in sidebars.

== Description ==
Shortcodes in Sidebar plugin allow shortcodes to execute when used in sidebar text widgets.  All that is needed is to download, install and activate. This is very useful for placing things like contact forms and other shortcode enabled features in sidebars.
		
== Installation ==

You can install Shortcodes in Sidebar Widgets directly from the WordPress admin panel. Visit the Plugins > Add New page and search for 'Shortcodes in Sidebar'. Click to install.  Once installed, activate and it is functional. 
	
Manual Installation Shortcodes in Sidebar Widgets plugin:

1. Download and unzip the file

2. Upload the full extracted folder to the `/wp-content/plugins/` directory

3. Activate the plugin through the 'Plugins' menu in WordPress.


You're done! The Plugin ready to use according to the directions in the description.


== Frequently Asked Questions ==

Q - Are there any settings?

A - No, The function is either activated or not.


Q - Why did you create a plugin for this function?

A - By having it as a plugin it works without needing to edit a themes functions file.The advantage is you can change your WordPress theme and your sidebar shortcodes will continue to work with needing to edit the theme functions.
